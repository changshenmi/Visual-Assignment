# 可视化仪表板项目  

一个交互式可视化仪表板，支持交互与决策边界实时展示。  

---

## 开发环境

推荐 node 版本 12.22.12
查看 node 版本

```shell
node -v
```

## 开发流程

安装依赖

```shell
npm install
```

## 打包发布
编译应用程序并将其置于 dist/ 目录：

```shell
npm run build
```
发布打包生成后的 dist/index.html

## 启动服务

```shell
npm run serve-watch #快速的编辑-刷新开发循环
# 或
npm run serve
```
